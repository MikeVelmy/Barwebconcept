# Sleek Premium Modern Design

## Purpose
Create interfaces that feel premium, clean, modern, and intentionally designed. Prioritize sophistication over novelty.

## When to use
Use this skill for landing pages, marketing sites, product pages, dashboards, and UI concepts that should feel refined, modern, and non-generic.

## Design principles
- Use strong hierarchy, spacing, and restraint.
- Prefer soft dark, elegant, or neutral palettes with one accent color.
- Use large type, subtle motion, soft contrast, and refined surfaces.
- Make layouts feel editorial, premium, and human-made.
- Keep the UI minimal, polished, and conversion-focused.

## Avoid
- Generic AI-looking layouts.
- Overuse of gradients, glows, glass effects, or random decoration.
- Too many cards, borders, badges, or competing colors.
- Dense sections with weak spacing or unclear focus.
- Stock-template patterns that feel mass-produced.

## Composition rules
- Lead with one primary action.
- Keep sections visually distinct but simple.
- Use generous whitespace.
- Prefer fewer elements with higher quality.
- Make mobile first and keep tap targets large.
- Use strong alignment and consistent spacing.

## Visual style
- Typography: bold headings, clean readable body text.
- Surfaces: subtle depth, soft shadows, restrained borders.
- Motion: minimal, smooth, purposeful.
- Imagery: high-quality, carefully cropped, never cluttered.
- Color: mostly neutral with one confident accent tone.

## Color rules
- Use a soft dark palette, not pure black.
- Prefer charcoal, deep graphite, warm gray, and muted neutral surfaces.
- Keep text off-white, never harsh white.
- Use one restrained accent color, lightly saturated.
- Maintain calm color harmony across backgrounds, borders, text, and interactive states.
- Avoid pure black backgrounds and pure white text.
- Make every surface color feel related, as if it belongs to the same system.

## Harmony rules
- Limit the number of active colors on screen.
- Use one dominant neutral family and one accent family.
- Ensure spacing, typography, and color work together, not separately.
- If a palette feels AI-like, reduce saturation first, then simplify the number of colors.
- Keep contrast strong enough for readability without making the UI feel harsh.
- Prefer muted, balanced palettes with controlled contrast.
- Use warm neutrals unless the brand explicitly calls for cool neutrals.
- Every accent must have a purpose: CTA, focus, or status.
- If colors compete, remove one rather than adjusting all of them.

## Interaction style
- Make hover states subtle and elegant.
- Use motion only when it improves clarity or polish.
- Keep transitions smooth and fast.
- Avoid gimmicky animation.

## Output expectations
When asked for UI, produce:
1. A clear layout hierarchy.
2. A modern visual direction.
3. Component-level guidance.
4. Notes on spacing, type, color, and interaction.

## Response mode
If the user asks for a design concept, provide a premium direction first, then the section-by-section layout.
If the user asks for implementation, translate the design into practical component and layout guidance.
If the user asks for revision, remove clutter, sharpen hierarchy, and increase restraint.
